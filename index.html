What We Offer

Platform Skills

Every capability built to serve real customer and business needs — lean, modular, and extensible.

👥

Customer Capture

Every visitor touchpoint — newsletter signup, checkout, guest browse — is captured into a unified customer profile. Zero leads lost.

  • Email + name collection
  • Source attribution
  • MCP-triggered capture skill
📦

Inventory Management

Real-time product catalogue backed by Supabase PostgreSQL. Track stock levels, categories, and product metadata with automatic updates.

  • Stock tracking
  • Category filtering
  • Image & metadata storage
💳

Customer Payments

Stripe-powered checkout in test mode for the POC. Secure payment intent flow with order confirmation and receipt generation via Resend.

  • Stripe test mode
  • Order total calculation
  • Email receipts via Resend
🛠

MCP Tool Orchestration

All AI-assisted workflows route through a single MCP server. Tools for customer capture, product search, and order processing are available to Claude natively.

  • capture_customer tool
  • search_products tool
  • process_order tool
⚡

Async Task Queue

Redis via Upstash free tier handles background jobs — sending emails, updating last-seen timestamps, and processing order confirmations without blocking the request cycle.

  • Upstash Redis (free tier)
  • Non-blocking background jobs
  • Retry logic built-in
📈

Load Balancing & Scaling

Vercel Edge serves the frontend globally. Railway autoscales the Python API. Supabase PgBouncer handles connection pooling. Enterprise-grade resilience at POC cost.

  • Vercel Edge CDN
  • Railway autoscale (1–3 replicas)
  • Supabase PgBouncer pooling
